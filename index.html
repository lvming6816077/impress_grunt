Yeoman

  • http://www.yeoman.io/
    •  

  • Yeoman scaffolds out a new application, writing your Grunt configuration and pulling in relevant Grunt tasks and Bower dependencies that you might need for your build.

Grunt

  • http://gruntjs.com/
    •  

  • Grunt is used to build, preview and test your project, thanks to help from tasks curated by the Yeoman team and grunt-contrib.

Bower

  • http://bower.io/
    •  

  • Bower is a package manager for the web. It offers a generic, unopinionated solution to the problem of front-end package management, while exposing the package dependency model via an API that can be consumed by a more opinionated build stack. There are no system wide dependencies, no dependencies are shared between different apps, and the dependency tree is flat.

Install them

*By Node.js Npm

Yeoman


Tips:

Make sure you have installed Node.js, Git and optionally, Ruby and Compass (if you plan to use Compass).


  • npm install -g yo
  • npm install -g generator-webapp

Usage:


  • yo webapp

    bower install underscore

    grunt

Grunt


Tips:

 ◇ This will put the grunt command in your system path, allowing it to be run from any directory.

 ◇ Note that installing grunt-cli does not install the Grunt task runner!

  • npm install -g grunt-cli

 ◇ The easiest way to add Grunt and gruntplugins to an existing package.json is with the command npm install --save-dev.

  • npm install grunt --save-dev

Usage:

  • yo webapp

    bower install underscore

    grunt

Bower


  • npm install -g bower

Usage:


     ◇ Using the dependencies listed in the current directory's bower.json

  • bower install

    •  ◇ Using a local or remote package

  • bower install "package"

    •  ◇ To search for packages registered with Bower:

  • bower search "name"

For example: Compile coffeescript

Compile CoffeeScript


     ◇ Install grunt plugin grunt-contrib-coffee

  • npm install -g grunt-contrib-coffee

    •  ◇ Config this task

  • coffee: {

          files: [{

            expand: true,

            cwd: '<%= yeoman.app %>/scripts',

            src: '{,*/}*.coffee',

            dest: '.tmp/scripts',

            ext: '.js'

          }]

    }

Less


     ◇ Install grunt plugin grunt-contrib-less

  • npm install -g grunt-contrib-less

    •  ◇ Config this task

  • less: {

          files: [{

            expand: true,

            cwd: '<%= yeoman.app %>/styles',

            src: '{,*/}*.less',

            dest: '.tmp/styles',

            ext: '.css'

          }]

    }

grunt task


     ◇ Config this task

  • grunt coffee

    grunt less

     ◇ build task

  • grunt.registerTask('build', [

                'clean:dist',

                'coffee',

                'less',

                ..

          ])
